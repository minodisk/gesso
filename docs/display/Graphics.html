cubicCurveTo(x1:Number, y1:Number, x2:Number, y2:Number, x3:Number, y3:Number):Graphics

Draws a cubic Bezier curve from the current drawing position to the specified anchor point. Cubic Bezier curves consist of two anchor points and two control points. The curve interpolates the two anchor points and curves toward the two control points.

x1:Number - Specifies the horizontal position of the first control point relative to the registration point of the parent display object.
y1:Number - Specifies the vertical position of the first control point relative to the registration point of the parent display object.
x2:Number - Specifies the horizontal position of the second control point relative to the registration point of the parent display object.
y2:Number - Specifies the vertical position of the second control point relative to the registration point of the parent display object.
x3:Number - Specifies the horizontal position of the anchor point relative to the registration point of the parent display object.
y3:Number - Specifies the vertical position of the anchor point relative to the registration point of the parent display object.
curveTo(x1:Number, y1:Number, x2:Number, y2:Number):Graphics

Draws a quadratic Bezier curve using the current line style from the current drawing position to (anchorX, anchorY) and using the control point that (controlX, controlY) specifies. The current drawing position is then set to (anchorX, anchorY). If the movie clip in which you are drawing contains content created with the Flash drawing tools, calls to the curveTo() method are drawn underneath this content. If you call the curveTo() method before any calls to the moveTo() method, the default of the current drawing position is (0, 0). If any of the parameters are missing, this method fails and the current drawing position is not changed.

x1:Number - A number that specifies the horizontal position of the control point relative to the registration point of the parent display object.
y1:Number - A number that specifies the vertical position of the control point relative to the registration point of the parent display object.
x2:Number - A number that specifies the horizontal position of the next anchor point relative to the registration point of the parent display object.
y2:Number - A number that specifies the vertical position of the next anchor point relative to the registration point of the parent display object.
